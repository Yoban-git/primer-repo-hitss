orden de los comandos
git status para ver el estado actual del repositorio
git add (. o nombre_archivo) prepara todo lo que le indiquemos para subirlo
git commit -m "mensaje" sube todos los cambios al repositorio
git pull o git pull --no-rebase origin master  baja los cambios echos
git push sube los cambios al repositorio
